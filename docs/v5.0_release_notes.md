# v5.0 版本宣告 — AI Prompt Generator

**發布日期：** 2026-02-16

## 🎉 重大改版：全新分頁式介面

### 新功能

| 功能 | 說明 |
|------|------|
| **分頁導航** | 6 大分類標籤：基本、外觀、動作、風格、環境、攝影 |
| **單選模式** | 每個區塊只能選一個選項，避免衝突 |
| **自訂按鈕** | 每個區塊都有「✏️ 自訂」按鈕，可輸入自定義值 |
| **性別切換** | 男/女髮型自動切換 |
| **髮色色票** | 18 種髮色圓形色票選擇器 |
| **眼色選擇** | 左右眼分別選色，支持異色瞳 |
| **新增選項** | 種族、職業、場景、天氣、鏡頭效果等 |

### 新增資料分類

- **基本 (Basic)**：種族、職業
- **外觀 (Appearance)**：髮型（男/女）、髮色、眼色、服裝
- **動作 (Action)**：表情、情緒
- **風格 (Style)**：動漫風格、藝術風格、藝術家、畫質
- **環境 (Environment)**：場景、天氣、光影
- **攝影 (Camera)**：角度、構圖、焦段、光圈、鏡頭效果

### 保留功能
- ✅ 音效系統（懸停、點擊、成功）
- ✅ 圖片懸停預覽
- ✅ 設定面板（AI 網站、背景圖）
- ✅ 自訂欄位
- ✅ 語言切換（中/英）
- ✅ 格式切換（Plain / YAML）

### 移除功能
- ❌ 拖放排序（改為分頁式導航）
- ❌ 折疊式面板（改為分頁切換）
- ❌ 多選標籤（改為單選）

### 修改檔案
- `script.js` — 完全重寫資料模型和渲染邏輯
- `index.html` — 新增 tab-bar / tab-content 容器
- `styles.css` — 新增 277 行分頁/標籤/色票/性別切換樣式
